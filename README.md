# 🥁 SoundScape - Virtual Drum Machine

A premium, interactive drum machine built with **React.js** and **Vite**. This project features a modern Synthwave/Neon aesthetic with full keyboard support and mobile responsiveness.

## ✨ Features
- **Neon UI:** High-end glassmorphism design with glowing effects.
- **Multi-Input:** Play sounds using Mouse Clicks or Keyboard keys (Q, W, E, A, S, D, Z, X, C).
- **Audio Control:** Real-time volume slider.
- **Fully Responsive:** Optimized for Desktop, Tablet, and Mobile screens (3x3 grid).
- **No External API:** Built using static assets for maximum reliability and speed.

## 🛠️ Tech Stack
- **React.js**
- **Vite** (for fast development)
- **Vanilla CSS** (Custom Design System)

## 🚀 How to Run Locally
1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
